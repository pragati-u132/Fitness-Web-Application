Inclusive Fitness Web Application
A web-based platform designed to provide personalized exercise routines, promoting accessibility and inclusivity for all users. The website features a BMI calculator that suggests exercises based on the userâ€™s BMI category. For users with disabilities, it includes additional features like text-to-speech guidance and tailored exercise recommendations specific to their physical condition (e.g., visual impairments, limited mobility, or other challenges).

Key Features:

BMI Calculator: Calculates BMI and provides personalized fitness suggestions.
Accessibility: Text-to-speech, shortcut keys, and font size adjustments for better usability.
Disability-Specific Routines: Offers safe, customized exercises for users with physical disabilities.
Tech Stack:

Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB
